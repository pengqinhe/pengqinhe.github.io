# Academic Homepage

This is a lightweight static personal homepage for an academic or scholarship profile.

## Edit Your Content

- Replace `Your Name`, position, affiliation, interests, projects, and paper entries in `index.html`.
- Replace `your.email@example.com` with your real email address.
- Replace `assets/profile-placeholder.svg` with your own photo. Keep the same filename or update the `img` path in `index.html`.
- Add links for CV, Google Scholar, GitHub, project pages, PDFs, and code.

## Preview

Open `index.html` directly in a browser. No build step is required.
